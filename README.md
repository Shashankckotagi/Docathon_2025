AI Agent for Early Screening of Gastrointestinal Diseases

GutSense is an AI-driven clinical assistant designed to support doctors in identifying gastrointestinal diseases at an early stage. The system analyzes patient data, predicts the likelihood of multiple GI conditions, and provides transparent reasoning behind every result.

A human-in-the-loop flow ensures that doctors review and validate all AI-generated outputs before patients receive any recommendations, keeping clinical safety at the center.
