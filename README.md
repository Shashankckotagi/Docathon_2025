AI Agent for Early Screening of Gastrointestinal Diseases

GutSense is an AI-based clinical assistant that predicts the risk of multiple gastrointestinal conditions from patient data.
The system uses XGBoost as the core machine learning model and provides transparent explainability for every prediction.

A doctor-in-the-loop workflow ensures that all AI results are reviewed, validated, and approved by a medical professional before reaching the patient.
